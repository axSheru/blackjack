# ACERCA DE  

En este proyecto se puede jugar Blackjack o 21. Está desarrollado en JS.  

## Reglas  

Gana el jugador que, tras sumar los puntos de sus cartas, esté más cerca de obtener 21 puntos sin pasarse.
